RemindMe 项目依赖文档

一、文档说明
本文档用于说明RemindMe待办提醒项目的开发环境、运行环境、依赖库、构建工具及相关配置要求，便于项目部署、运行、维护与验收。

本项目为基于JavaFX图形用户界面的个人待办事项管理程序，支持任务新增、编辑、删除、完成状态管理、提醒功能、日历视图以及JSON数据持久化。

二、开发与运行环境
1. 开发语言
Java

2. JDK 版本
JDK 17

3. 图形界面框架
JavaFX 21.0.2

4. 构建工具
Maven

5. 开发工具
IntelliJ IDEA

6. 操作系统
开发环境：Windows
运行环境：Windows

7. 打包工具
jpackage
用于生成 Windows 平台可执行程序或 app-image 发布目录。

三、项目核心依赖库
1. JavaFX Controls
依赖名称：org.openjfx:javafx-controls
版本：21.0.2
用途：提供按钮、标签、列表、对话框、布局容器等图形界面控件，是本项目界面开发的核心依赖。

2. JavaFX FXML
依赖名称：org.openjfx:javafx-fxml
版本：21.0.2
用途：提供 FXML 相关支持，用于 JavaFX 项目的界面扩展能力。

3. Gson
依赖名称：com.google.code.gson:gson
版本：2.10.1
用途：用于任务数据的 JSON 序列化与反序列化，实现本地数据持久化存储。

四、运行要求
1.首次运行前应保证：
（1）项目根目录下存在 usrdata 文件夹；
（2）usrdata/data.json 可被正常读取和写入；
（3）若文件不存在，程序应允许自动创建或初始化。

2.运行前应保证：
（1）项目根目录下存在 icons 文件夹；
（2）所需 png/ico 图标文件完整存在；
（3）若缺失图标文件，程序可能无法正常显示按钮图标、窗口图标或 exe 图标。


在实际运行时，应确保依赖库完整、运行环境正确、数据目录和图标资源目录配置齐全，以保证程序能够正常启动、正常显示界面并完成数据读写。
